## ❓ Troubleshooting Common Issues

**tengu does not start:**  
Check if your operating system meets the requirements. Make sure the installation completed without errors. Restart your computer and try again.

**Scan fails or is slow:**  
Ensure your internet connection is stable. Reduce scan depth in settings if scan is too slow. Close other applications to free system resources.

**No results after a scan:**  
Verify the IP address or domain you entered is correct and reachable. Try running a basic network ping test on your target before scanning.

**Updates fail:**  
Check your internet connection. Run tengu as an administrator to allow updates. You can also download the installer again from the GitHub page and reinstall.
